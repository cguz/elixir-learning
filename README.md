# Elixir Learning

## Create Mix Project

    $ mix new chatbotserver

## Run command 

    $ elixir lib/chatbotserver.ex

    Compiled into bytecode (in memory) and then run on Erlang virtual machine

### Interactive Shell

    $ iex 
    $ iex> c "lib/chatbot.ex"
    $ iex> ChatbotServer.hello("Martin")

or 

    $ iex lib/chatbotserver.ex

or 

    $ iex -S mix 

## Recompile

    $ iex> r ChatbotServer
